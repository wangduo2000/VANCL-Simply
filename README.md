# VANCL-Simply
create a new simply website of vancle
use node as server to running this website
use a user.json as user database

## To start

This is a demo for Vancl

``` bash
# serve at localhost:2333
node router.js

```

## Folder structure
* router.js
* user.json
* src -your app
  * css
  * html
  * image
  * js
    * common
    * js
    * lib
